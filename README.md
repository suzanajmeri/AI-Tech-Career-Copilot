# AI Tech Career Copilot

AI Tech Career Copilot is an AI-powered career guidance web application that helps users explore technology careers, discover relevant skills, generate personalized learning roadmaps, and interact with an intelligent chatbot for career support.

## Features

- Career Recommendation System
- AI Career Chatbot
- Skill Explorer
- Personalized Career Roadmaps
- Learning Resource Suggestions
- Voice Response Support
- Responsive User Interface

## Technologies Used

- Python
- Flask
- HTML5
- CSS3
- JavaScript
- Flask-CORS

## Functionalities

### Career Recommender
Users can enter skills like Python, SQL, React, AI, and more to receive suitable technology career suggestions.

### Career Roadmap Generator
Generates step-by-step learning roadmaps for careers such as:
- AI Engineer
- Data Scientist
- Backend Developer
- Frontend Developer

### Skill Explorer
Allows users to explore:
- Related careers
- Learning resources
- Technology guidance

### AI Chatbot
Users can interact with the chatbot using queries like:
- career for python
- roadmap for ai engineer
- explore sql

## Installation

### Clone Repository
```bash
git clone https://github.com/your-username/AI-Tech-Career-Copilot.git
```

### Move into Project Folder
```bash
cd AI-Tech-Career-Copilot
```

### Install Dependencies
```bash
pip install flask flask-cors
```

### Run Application
```bash
python app.py
```
